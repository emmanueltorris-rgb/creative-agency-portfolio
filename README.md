# Creative Agency Portfolio Platform

A mordern, responsive React application built to showcase project work and dynamically manage a portfolio

## Features

### Dynamic Portfolio:
Add new projects vis a real-time form.
### Instant Search:
Filter projects by name using a high-perfomance search bar.
### Responsive Design:
Optimized for mobile, tablet, and desktop using css grid.
### Tested Logic:
Unit tests included to ensure search and state reliability.

## Tech Stack
### Frontend
React 19, Vite
### Testing
Vitest, React Testing Library
### Environment
Ubuntu (Development)
